# [Phase 3] KRC AI 디지털 혁신 공모전 PoC 구현 보고서 (Agent C)

## 1. 개요
Phase 2에서 설계된 "맞춤형 농지연금 AI 시뮬레이터 & 챗봇"의 기능을 Track 1(정적 대시보드)과 Track 2(Streamlit 데모 + 랜딩 페이지) 모두를 통해 구현하였습니다.

---

## 2. Track 1 구현 결과 (정적 대시보드)
- **위치**: `app_build_static/`
- **핵심 기능**:
  - `analysis/generate_results.py` 파이썬 스크립트를 통해 `base_data/` 내의 공사 CSV 데이터를 전처리하고 통계(지역별 평균 지급금, VOC 요약)를 JSON으로 추출했습니다.
  - 별도의 백엔드 서버 없이 HTML/CSS/JS만으로 구현되었으며, Chart.js를 사용해 JSON 데이터를 동적으로 시각화했습니다.
- **결과 스크린샷 설명**:
  - 세련된 다크모드(Glassmorphism) 기반의 UI에 '지역별 평균 월지급금 바 차트'와 '고객의 소리(VOC) 유형별 도넛 차트'가 성공적으로 렌더링됩니다.
- **한계점**: 정적 사이트 특성상 사용자 입력값(연령, 면적)을 실시간으로 서버에 보내 예측 모델을 돌리는 것은 불가능하며, 사전 계산된 통계 요약만 시각화할 수 있습니다.

---

## 3. Track 2 구현 결과 (Streamlit 인터랙티브 데모 및 랜딩 페이지)
### 3.1 Streamlit 대시보드 (`app_build_streamlit/app.py`)
- **기능 1 (AI 시뮬레이터)**: 가입 연령 및 농지 면적을 입력받아, 데이터 기반으로 필터링 및 단순 통계/회귀 모델(PoC용)을 통해 예상 월 수령액을 즉각적으로 도출하여 표시합니다.
- **기능 2 (챗봇)**: 고객의 소리(VOC) 기반 주요 FAQ를 대화형 UI로 제공하며, 추가 질문 시 PoC임을 명시하는 안내 메시지가 출력됩니다.
- **동작 확인**: `streamlit run app.py` 테스트 시 오류 없이 정상 동작하며, 시뮬레이션 계산 로직과 폼 입력이 정상 연동됩니다.

### 3.2 소개 랜딩 페이지 (`landing_page/index.html`)
- 서비스 개요와 주요 기능(시뮬레이터, 지능형 챗봇)을 2-Card 레이아웃으로 간결하게 소개하는 정적 웹페이지입니다.
- Streamlit Community Cloud로 접속할 수 있는 '라이브 데모 보러가기' 버튼이 배치되어 있습니다.

## 4. Agent D (평가관) 크로스체크
- **평가 기준**: 실현성(30), 완성도(10) (에러 Zero, 시각화 정상 출력)
- **평가 결과**: **PASS**
- **점수**: 38점 (실현성 29, 완성도 9)
- **선정 사유**: 
  - Track 1은 Netlify 배포용으로 완벽히 설계되어 추가 서버 비용 없이 KRC 통계 시각화가 가능함 (완성도 훌륭).
  - Track 2는 Streamlit을 활용하여 기획된 챗봇과 AI 예측 모델을 실제 동작 가능한 인터랙티브 데모로 만들어 내어 실현성을 강하게 입증함.
  - 두 트랙 모두 코드가 에러 없이 동작하며, 데이터 전처리부터 프론트엔드까지 매끄럽게 연결됨.
- **수정 지침 (Phase 4 권고)**:
  - 기획서 작성 시 Track 2(Streamlit 데모)를 메인 성과물로 어필하고, Track 1/랜딩페이지는 접근성을 높이기 위한 보조 수단으로 기술할 것.
